# Easee Charge

> **This repository is archived.** The app has been sold and is now an official Homey app maintained by Athom. For the latest version, please install [Easee](https://homey.app/a/no.easee/) from the Homey App Store.

---

This repository contains the original community-developed Easee Charge app for Homey. It added support for the Easee charger and Easee Equalizer, allowing users to:

- Start and stop charging on a schedule or based on flow conditions
- Monitor energy usage via the Easee Equalizer
- Access charging statistics
- Control smart charging features

Thank you to everyone who used and supported this app over the years!
